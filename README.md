
# Codeant Assignment

    
## Run Locally

Clone the project

```bash
  git clone https://github.com/vaishnavi-waghmare/Assignment.git
```

Go to the project directory

```bash
  cd Assignment
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

```bash
  localhost:3000
```


## Tech Stack

**Framework:**  Next.js

**Ui Framework:** shadcn, TailwindCSS
